# Introduction

This set of modules will walk you through the documentation for kubernetes and ask you to setup various components of the kubernetes system locally. In order to do anything you will need to setup kubernetes locally using [minikube](https://github.com/kubernetes/minikube).

## The Test

1. Setup minikube locally and launch hello-minikube following the readme
2. Load the kubernetes dashboard from minikube in your browser
3. Find the hello-minikube pod running
